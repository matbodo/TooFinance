# TOO Finance

A modern and responsive landing page for TOO Finance, a fintech focused on credit, insurance, financial planning, and payment solutions for the healthcare ecosystem.

The project presents financial solutions for clinics, hospitals, healthcare professionals, pharmacies, industries, educational institutions, and patients. The interface was designed with a focus on clarity, conversion, responsiveness, and clean code organization using React and TypeScript.

## Technologies

- React
- TypeScript
- CSS
- Vite

## Main Sections

- Institutional hero section
- Healthcare ecosystem segments
- Financial solutions
- Quick simulator
- Key metrics
- Personas
- Testimonials
- Educational content
- Contact call-to-action
